JavaScript中的常见排序

快速排序

思想：找基数 → 左右各做大小排序 → 递归 → 拼接

时间复杂度：O(n log n)

选择排序

思想：找到最小（最大）值 → 放在左边（右边））→ 记住最小值的位置 → 缩短排序长度

时间复杂度：O(N²)

插入排序

思想：认为第一个元素有序 → 下一个元素从后向前扫描 → 在已排序的元素中，如果有大于新元素的，移动到后面位置 → 重复上一步骤，直到找到已排元素小于或者等于新元素 → 插入到其后面 → 重复上一组操作

时间复杂度：O(N²)
